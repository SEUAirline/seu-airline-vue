## 🎨 用户下拉菜单

### 菜单触发

**鼠标悬停显示:**
- 鼠标移入用户名区域 → 显示下拉菜单
- 鼠标移出 → 隐藏下拉菜单

### 菜单内容

```
┌─────────────────────┐
│ 👤 个人中心         │
│ 📋 我的订单         │
│ 👥 常用旅客         │
│ ⚙️  账号设置         │
│ ─────────────────── │
│ 🚪 退出登录         │
└─────────────────────┘
```

**菜单项:**
1. **个人中心** - 跳转到 `/user/profile`
2. **我的订单** - 跳转到 `/user/orders`
3. **常用旅客** - 跳转到 `/user/passengers`
4. **账号设置** - 跳转到 `/user/settings`
5. **退出登录** - 执行退出操作

## 🎯 交互特性

### 1. 悬停效果

**按钮:**
- 默认：灰色文字
- 悬停：蓝色文字
- 图标：用户头像图标 + 下箭头

**菜单项:**
- 默认：白色背景
- 悬停：浅蓝色背景 + 蓝色文字
- 退出按钮：悬停时红色背景

### 2. 视觉设计

**菜单样式:**
- 白色背景
- 圆角边框
- 阴影效果
- 边框线分隔

**图标:**
- 每个菜单项都有对应的图标
- 图标 + 文字的组合
- 统一的间距和对齐

## 💻 代码实现

### HTML 结构

```vue
<div class="relative" @mouseenter="showUserMenu = true" @mouseleave="showUserMenu = false">
  <!-- 触发按钮 -->
  <button class="flex items-center space-x-2">
    <svg>用户图标</svg>
    <span>{{ userInfo?.username }}</span>
    <svg>下箭头</svg>
  </button>
  
  <!-- 下拉菜单 -->
  <div v-show="showUserMenu" class="absolute right-0 mt-2">
    <router-link to="/user/profile">个人中心</router-link>
    <router-link to="/user/orders">我的订单</router-link>
    <router-link to="/user/passengers">常用旅客</router-link>
    <router-link to="/user/settings">账号设置</router-link>
    <div class="border-t"></div>
    <button @click="handleLogout">退出登录</button>
  </div>
</div>
```

### TypeScript 状态

```typescript
// 用户菜单显示状态
const showUserMenu = ref(false)
```
